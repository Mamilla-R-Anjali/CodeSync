# CodeSync Features

## Phase 1 - MVP

- Create coding rooms
- Join existing rooms
- Real-time collaborative editing
- Basic syntax highlighting

## Phase 2

- User authentication
- Save and reopen coding sessions
- Room history
- Multiple programming languages

## Phase 3

- Compile and run code
- Input and output console
- Error highlighting
- Auto-save

## Phase 4

- Voice chat
- Video chat
- Screen sharing
- AI code suggestions
- Code formatting

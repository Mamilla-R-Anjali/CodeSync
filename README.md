# CodeSync

CodeSync is a real-time collaborative code editor that allows multiple users to write and collaborate on code in shared coding rooms.

## Project Vision

The goal of CodeSync is to build a platform where developers can:

- Create coding rooms
- Invite other users
- Collaboratively edit code in real time
- Run code
- View output
- Save code versions

## Planned Technologies

- React.js
- Node.js
- Express.js
- WebSockets
- MongoDB
- Docker
- AWS
